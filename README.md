## HOW TO LAUNCH
1. run ```npm i```
2. create .env file in /backend
3. insert ```PORT=8000
DB_FILE=db/dev.db
JWT_SECRET=SomeSecretCode
JWT_EXPIRES_IN=SomeNumber``` into the .env (in backend)
4. run ```npm run migrate && npm run seed```
5. cd into /backend
6. run ```npm start```
7. cd into /frontend
8. run ```npm run dev```
